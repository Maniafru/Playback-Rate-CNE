Thanks for installing this! 

I made this for a mod called FNF: RANDOM (WIP)

I made it because i realized CNE does not have playback rate like PE does, (or i think they dont)

So i decided to make playback rate option in the options tab!

WHERE TO PUT: put global.hx in your mods/ data folder.
put options in your mod/data/config
and put playbackrate.hx in your mod/songs/your-spng-name/scripts (you might need to make a new folder for it) 
and paste the script in there. dont mess with that script

HOW TO USE: you can change how fast or slow you want your min or max speed to be in your settings. 
For example, in the options.xml you can change the min to be 0.1, 0r 0.5 to be slow, or 3 to be fast.

And thats it ! Have fun!
